# 風格參考用例

本檔案提供 **Laike 台灣區內容種草技能** 的文風參考。
用途是幫助模型理解「台灣繁體中文、Plurk 語感、二次元圈層」的寫法。

## 使用原則

1. 這些是**文風參考**，不是固定模板。
2. 產出時要避免照抄，應依照當次主題、角色、情緒重新生成。
3. 若內容涉及推薦、試用、邀請碼，必須自然且清楚，不可誤導讀者。
4. 文案應像真實社群分享，但**不可假造個人經歷**。
5. 每篇建議控制在 20–200 字之間，文末附 1–2 個 Tag。

## A 類：IP 代餐／夢女向情緒分享

### 文風特徵
- 語氣像在碎碎念、哀號、發瘋
- 核心情緒是「官方不夠吃、我需要代餐」
- 可帶一點暈船、沉浸感、角色陪伴感
- 不可寫成硬廣

### 參考方向
- 抱怨卡池、劇情、體力、互動太少
- 表達「最近改去別的地方補情緒價值」
- 若提到 Laike，應像自然分享工具或體驗

## B 類：獵奇／反差／高張力設定分享

### 文風特徵
- 重点是「這設定也太扯」「怎麼會這麼香」
- 可以危險、反差、病嬌感，但不可露骨色情
- 像在分享最近看到一個很有討論度的人設

### 參考方向
- 強烈控制慾
- 病嬌式執著
- 表面冷淡、實際失控
- 背德感與張力，但只點到為止

- 假裝真實受害或真實危險經驗來引流

### 文風特徵
- 像語C玩家、OC 創作者在推自家角色
- 強調人設亮點、氣質、互動感
- 可附簡短「食用說明書」
- 適合搭配角色表內容，但不可照抄表格欄位

- 嘴壞但會默默護人的類型

## D 類：玄學／情緒樹洞／深夜低潮

### 文風特徵
- 語氣脆弱、內斂、像深夜自言自語
- 可以是失眠、失戀、內耗、很想被理解
- 若提到功能，應像在分享「最近試過一個方式」
- 適合塔羅、傾聽、陪伴、慢慢聊

### 參考方向
- 深夜 emo
- 覺得自己很亂、很累
- 想找地方說話但不想打擾朋友
- 想問感情／運勢／關係卡點

### 正向示例 1
最近剛失戀嘛 找了個玄學軟體問看看
結果六爻給我的卦象
直接說這段感情「緣淺情薄 不必強求」

想說會不會太剛好 再試占星
結果它分析說我最近情緒波動大
是因為金星跟土星有相位
這段時間適合整理情感 不適合挽回

⋯⋯⋯所以玄學也叫我放生他？？
好吧 两個功能都講一樣的話
我還是乖乖聽話好了 😭

有沒有人也想玩的 覺得滿準的 
#情緒樹洞 #AI聊天


### 避免示例
- 誇大療效
- 把產品說成心理治療替代品
- 假裝嚴重個案來博同情
- 過度強調「超準」「百分之百有效」

## 生成時的自我檢查

每次產出前後都要確認：

1. 是否為台灣繁體中文
2. 是否保有 Plurk 的碎碎念感
3. 是否像真人分享，而不是品牌通稿
4. 是否沒有誤導、假造體驗、假冒身分
5. 是否沒有明顯中國大陸用語
6. 是否有 1–2 個合理 Tag
7. 是否保留二次元／乙女／語C圈層語感
